# Portafolio Personal - Carlos Miguel Barbosa Torres 🚀

Este repositorio contiene el código fuente de mi portafolio web personal, diseñado para mostrar mis proyectos, habilidades y experiencia como Desarrollador Full-Stack Junior.

## 📖 Acerca del Proyecto

Es una página web estática de tipo _landing page_ (una sola página) completamente responsiva, construida desde cero. El diseño se enfoca en una interfaz limpia, simétrica y moderna, con atención a los detalles visuales como efectos de resplandor (glowing) y maquetación precisa utilizando CSS Grid y Flexbox.

Como estudiante de séptimo semestre de Ingeniería de Sistemas, este portafolio refleja mi capacidad para llevar un diseño de Figma a código real, estructurado y mantenible.

## 🛠️ Tecnologías y Herramientas Utilizadas

- **HTML5:** Estructura semántica del contenido.
- **CSS3:** Estilos personalizados, animaciones, variables y diseño responsivo (_Mobile First_ / _Desktop_).
- **JavaScript:** Interactividad básica.
- **Git & GitHub:** Control de versiones.
- **Figma:** Interpretación e implementación del diseño UI/UX.

## ✨ Características Principales

- **Diseño 100% Responsivo:** Adaptable a pantallas móviles (320px+), tablets y monitores de escritorio (hasta 1440px).
- **CSS Grid y Flexbox:** Uso de metodologías modernas de maquetación para asegurar una alineación perfecta de los elementos sin importar la resolución.
- **Efectos Visuales Nativos:** Implementación de gradientes radiales e interacciones _hover_ personalizadas en botones y tarjetas de proyectos.
- **Sección de Proyectos:** Galería estructurada para destacar proyectos formativos y reales (ej. _Landing Page de Cafetería_).

## 🚀 Uso

Este portafolio está diseñado para ser visualizado directamente en cualquier navegador web. Para explorar la página:

- Descarga o explora los archivos de este repositorio y abre el archivo `index.html` haciendo doble clic sobre él. Tu navegador por defecto (Chrome, Safari, Edge, etc.) abrirá el portafolio automáticamente.
- Si deseas explorar el código, puedes abrir la carpeta en tu editor de texto favorito (como Visual Studio Code) y utilizar extensiones como _Live Server_ para previsualizar los cambios en tiempo real.
